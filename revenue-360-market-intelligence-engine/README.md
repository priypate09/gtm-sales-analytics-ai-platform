# Revenue 360 Market Intelligence Engine

Round 1 setup scaffold for a multi-agent revenue intelligence workflow.

## Quick start

1. Create and activate a Python 3.11 virtual environment.
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Generate synthetic and sample data:
   - `python scripts/generate_synthetic_data.py`
4. Run setup verification:
   - `python scripts/verify_setup.py`

## Current scope

- Folder and file scaffolding
- Config and environment placeholders
- Deterministic synthetic CSV generation
- Basic setup verification checks
